# 面试题库

> 问题页不存完整答案；通过 `answer_note` 进入独立答案页。

## 本人真实面试题

```dataview
TABLE source_company AS "公司", source_role AS "岗位", self_occurrences AS "遇到次数", prediction_hit AS "预测命中", link(answer_note, "查看答案 →") AS "答案"
FROM "03 面试题库"
WHERE type = "question" AND source_type = "self-real"
SORT last_seen DESC
```

## 到期复习

```dataview
TABLE source_type AS "来源", category AS "题型", last_score AS "得分", next_review AS "复习日", link(answer_note, "查看答案 →") AS "答案"
FROM "03 面试题库"
WHERE type = "question" AND status != "mastered" AND (!next_review OR next_review <= date(today))
SORT last_score ASC, difficulty DESC
```

外部真题按岗位归纳在 [[03 面试题库/_真题库索引]]。
