# 面试自生长库：使用顺序

这座库只有一条主线：

**真实简历 → 新 JD → 能力与匹配 → 简历更新方案 → 预测题 + 独立答案 → 每日模拟 → 真实面试回流。**

## 0. 首次启用 Obsidian 功能

1. 在“核心插件”中启用 **模板** 和 **日记**；模板目录与日记目录已由本库预设。
2. 建议安装社区插件 **Dataview**，用于自动生成岗位、问题、能力、故事和简历索引。
3. 如果希望在 Obsidian 内通过命令新建模板笔记，安装社区插件 **Templater**；不安装也可以让 Codex 按模板直接创建文件。
4. 在 Codex 中打开这座库，调用 `$obsidian-interview-growth-vault`，或复制 [[99 系统/AI 工作台]] 中对应流程。

## 1. 首次导入简历

1. 把 PDF、Word 或截图放进 `07 简历库/原始文件`。
2. 打开 Claudian/Codex，发送 [[99 系统/AI 工作台#流程 A：首次导入简历|流程 A]]。
3. AI 会整理 [[07 简历库/01 当前通用简历]] 和 [[07 简历库/00 简历事实底稿]]。
4. 按一次一个问题确认经历、数字和贡献边界。事实底稿是以后所有匹配与行为题答案的唯一真实来源。

## 2. 导入并分析 JD

JD 支持三种输入：直接粘贴文字、拖入截图、打开 Web Clipper 笔记。发送 [[99 系统/AI 工作台#流程 B：导入一个新 JD 并完整处理|流程 B]] 后，AI 会依次完成：

1. 查 [[01 岗位库/_投递清单]] 防止重复投递。
2. 提取职责、能力、Base、办公方式、薪资和福利。
3. 使用 `$obsidian-interview-growth-vault` 做简历证据映射与更新方案。
4. 使用 `$obsidian-interview-growth-vault` 结合真题库、JD、简历事实和故事库生成 Top 20 题；岗位页点击题目文字即可进入答案。
5. 将问题放入 `03 面试题库`，答案另存入 `04 答案库/题目答案`。

Web Clipper 默认不下载网页图片；图片型 JD 请直接附图，或运行 `Download attachments for current file`。

## 3. 手动标记已经投递

打开 [[01 岗位库/_投递清单]]，亲手点击岗位前的复选框：

- 未勾选：还没有投递，可以继续准备或投递。
- 已勾选：已经投递，以后不再推荐重复投递，但会优先进入模拟面试。

AI 只能读取状态，不能替你勾选。

## 4. 每日模拟面试

点击左侧日历中的今天，或运行 `Daily notes: Open today's daily note`。每日笔记会自动出现“完成一次模拟面试”的提醒。发送 [[99 系统/AI 工作台#流程 D：今日模拟面试|流程 D]] 后，AI 会优先使用已投递公司的题目，一次问一道。

问题页不会展示答案；你回答之后，AI 才读取独立答案页进行反馈。

## 5. 保存真实面试问题

面试结束后用 `99 模板/真实面试记录模板` 创建记录，先原样写下问题、追问和自己的回答，再发送 [[99 系统/AI 工作台#流程 C：录入一次真实面试并校准预测|流程 C]]。AI 会保留原始记录、把问题归档去重，并记录预测命中或预测外新题。

## 6. 导入小红书等平台的真题截图

1. 直接把截图拖入对话或 Obsidian，并尽量附上平台、公司、岗位和原帖链接。
2. 发送 [[99 系统/AI 工作台#流程 G：导入外部真题截图|流程 G]]。
3. AI 会保留原图、忠实转写、拆题去重，并优先生成答案。
4. 外部网帖标为 `external-real`；它会参与预测，但不会被冒充为公司官方题库。
5. 以后你本人面试遇到同题，系统会将其升级为 `self-real`，记录预测命中并提高相同岗位的准备优先级。

## 7. 每周让库继续生长

每周执行一次 [[99 系统/AI 工作台#流程 E：每周自生长整理|流程 E]]，归纳相似岗位、合并重复题、寻找能力与故事缺口。

## 文件夹职责

| 文件夹 | 唯一职责 |
|---|---|
| `00 临时收集表` | 尚未处理的文字、链接和零散信息 |
| `01 岗位库` | JD、岗位分析和手动投递清单 |
| `02 能力库` | JD 能力要求与个人证据 |
| `03 面试题库` | 保存问题、考察点、练习记录和外部真题来源页，不存完整答案 |
| `04 答案库/题目答案` | 问题对应的独立参考答案/个人答案 |
| `04 答案库/故事库` | 真实行为故事和 BQ Story Bank |
| `05 面试记录` | 真实面试原始记录与模拟面试记录 |
| `06 每日训练` | 每日提醒、模拟过程和复盘 |
| `07 简历库` | 事实底稿、通用简历和各 JD 独立定制版本 |
| `98 附件` | 图片和附件；真题截图集中放在 `98 附件/真题截图` |
| `99 模板` | 新建笔记模板 |
| `99 系统` | 工作流、字段与 AI 指令 |
