# 字段规范

YAML 字段使用英文是为了让查询稳定，正文继续使用中文。

| 字段 | 含义 | 推荐值 |
|---|---|---|
| `type` | 笔记类型 | `job`、`skill`、`question`、`answer`、`mock-interview`、`daily-practice` |
| `role_family` | 岗位族 | 产品经理、数据分析、前端开发等稳定名称 |
| `base_location` | 岗位 Base 地 | 上海、深圳、北京；不明确时留空 |
| `work_mode` | 办公方式 | `onsite`、`hybrid`、`remote` 或留空 |
| `salary_range` | JD 原始薪资范围 | 如 `25k–35k`、`面议`；保留原单位 |
| `daily_salary` | JD 明确披露的日薪 | 仅在实习/兼职等按日计薪时填写，不从月薪推算 |
| `salary_months` | 薪资月数 | 如 `13`、`14–16`；未说明时留空 |
| `benefits` | 其他待遇 | 奖金、股票、餐补等 YAML 列表 |
| `status`（岗位） | 投递进度 | `collecting`、`applied`、`interviewing`、`offer`、`closed` |
| `status`（题目） | 掌握状态 | `new`、`draft`、`practicing`、`mastered` |
| `category` | 题型 | `behavioral`、`technical`、`case`、`motivation`、`resume`、`reverse` |
| `difficulty` | 难度 | 1–5 |
| `proficiency` | 当前熟练度 | 1–5 |
| `priority` | 优先级 | `high`、`medium`、`low` |
| `type: resume-fact-base` | 简历事实底稿 | 只保存可验证事实和确认状态 |
| `type: resume-master` | 当前通用简历 | 当前对外表达，不由 AI 直接覆盖 |
| `type: resume-version` | JD 定制版本 | 每个公司、岗位、JD 单独生成 |
| `status`（简历版本） | 版本阶段 | `needs-input`、`draft`、`reviewed`、`submitted`、`archived` |
| `match_score` 等 | 材料诊断分 | 必须附证据和扣分原因，不代表录用率 |
| `analysis_status` | JD 处理阶段 | `raw`、`jd-parsed`、`needs-resume`、`needs-input`、`ready` |
| `resume_match_score` | 简历与 JD 匹配诊断 | 仅在 JD 和简历都可解析时填写 |
| `source_type`（问题） | 当前最高证据等级 | `predicted`、`external-real`（平台转述真题）、`self-real`（本人真实面试）、`daily` |
| `source_records` | 原始来源笔记 | 指向 `03 面试题库/真题来源` 或 `05 面试记录` 的 YAML 列表 |
| `source_images` | 真题截图 | 指向 `98 附件/真题截图` 的 YAML 列表 |
| `occurrence_count` | 总出现次数 | 外部来源数 + 本人遇到次数；同一帖重复截图只计一次 |
| `external_occurrences` | 外部独立来源数 | 按帖子/来源记录去重计数 |
| `self_occurrences` | 本人面试遇到次数 | 以真实面试记录为依据 |
| `prediction_basis` | 预测依据 | JD、Hidden Signal、外部真题、本人面试、相似岗位等列表 |
| `prediction_confidence` | 预测置信级别 | `high`、`medium`、`low`；不是概率 |
| `prediction_hit` | 是否被本人面试验证 | `true` / `false`；只有本人真实面试可改为 `true` |
| `last_seen` | 最近出现日期 | `YYYY-MM-DD` |
| `answer_note` | 独立答案链接 | 必须指向 `04 答案库/题目答案` |
| `record_type` | 面试记录类型 | `real`、`mock` |
| `type: true-question-source` | 外部真题来源页 | 一张或一组同帖截图对应一页，保留 OCR、平台、公司、岗位与链接 |

规则：数组字段使用 YAML 列表；日期统一用 `YYYY-MM-DD`；没有信息时留空或标记“待确认”。薪资只记录原文明确披露内容，不自行推算年包。投递状态只认 [[01 岗位库/_投递清单]] 的手动复选框；问题页和答案页必须分开。
